# 🚀 DSA Algorithm Analyzer & Visualizer

AI-powered tool to analyze, debug, explain, and visualize Data Structures & Algorithms code.

## ✨ Features

* Analyze DSA code in **C++, Python, Java, and JavaScript**
* Detect bugs and suggest fixes
* Time & Space Complexity analysis
* Line-by-line code explanation
* Step-by-step algorithm visualization
* Play, Pause & Speed Controls
* Built-in demo algorithms

## 🛠️ Tech Stack

* React
* Vite
* Groq AI (`openai/gpt-oss-120b`)

## 🚀 Installation

```bash
git clone https://github.com/shainti/DSA-Algorithm-visualizer.git
cd DSA-Algorithm-visualizer
npm install
```

Create a `.env` file in the project root:

```env
VITE_GROQ_API_KEY=your_groq_api_key
```

Start the development server:

```bash
npm run dev
```

## 🔑 Get a Free Groq API Key

1. Visit https://console.groq.com
2. Sign up for a free account
3. Open **API Keys**
4. Create a new API key
5. Add it to your `.env` file
